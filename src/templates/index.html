What we built

End-to-End ML Pipeline

From raw CSV data to a live web prediction — every step automated, every model compared, best one deployed.

Data Ingestion

Reads 1,000 student records, splits into 80/20 train-test sets, and saves to artifacts automatically.

Data Transformation

Numerical features scaled with StandardScaler. Categorical features one-hot encoded. Preprocessor saved as .pkl.

Model Selection

7 models trained with GridSearchCV. Best model (Random Forest, R²=0.85) automatically selected and saved.

Instant Prediction

Loads saved model and preprocessor, applies identical transformations, returns prediction in milliseconds.

Custom Exception Handling

Every error includes the exact file name and line number — no silent failures anywhere in the pipeline.

Cloud Deployed

Deployed on Render with Gunicorn. Trains automatically on every deploy. Live at a public URL.
